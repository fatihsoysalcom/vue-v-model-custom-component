# Vue v-model Custom Component

This example demonstrates how to add `v-model` support to custom Vue.js components. It shows a parent component binding its data to a custom input component using `v-model`. The custom component accepts a `modelValue` prop and emits an `update:modelValue` event to achieve two-way data binding.

## Language

`html`

## How to Run

1. Save the code as `index.html`.
2. Open `index.html` in your web browser.
